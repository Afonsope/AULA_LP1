# ☕ Projetos - Linguagem de Programação I (Java)

Este repositório reúne os projetos desenvolvidos durante as aulas da disciplina **Linguagem de Programação I**, utilizando a linguagem **Java**.

O objetivo é registrar minha evolução na programação, praticar lógica de programação e aplicar os conceitos aprendidos em sala de aula.

## 📚 Conteúdos Praticados

Durante os projetos são utilizados conceitos fundamentais de programação, como:

* Estrutura básica de programas em Java
* Entrada e saída de dados com `Scanner`
* Estruturas condicionais (`if`, `else`, `switch`)
* Estruturas de repetição (`for`, `while`)
* Variáveis e tipos de dados
* Operadores lógicos e aritméticos
* Organização e lógica de algoritmos

## 📂 Estrutura do Repositório

Cada pasta ou arquivo representa um exercício ou projeto desenvolvido em aula.

Exemplos de projetos:

* Sistema de notas de alunos
* Sistema de votação
* Exercícios de lógica
* Programas de cálculo
* Pequenos sistemas interativos no terminal

## 🛠 Tecnologias Utilizadas

* **Java**
* **JDK**
* **VS Code / IDE Java**

## 🎯 Objetivo do Repositório

* Praticar lógica de programação
* Consolidar conhecimentos em Java
* Registrar minha evolução na programação
* Criar um histórico de projetos acadêmicos
